Package uk.co.mmscomputing.imageio.
sub-directories:
  1. uk.co.mmscomputing.imageio.bmp:
    Classes that enable the java imageio system to read and write *.bmp files.
  2. uk.co.mmscomputing.imageio.gif:
    Classes that enable the java imageio system to read and write *.gif files.
  3. uk.co.mmscomputing.imageio.jpeg:
    A few jpeg support classes for the tiff package. Compression mode 7.
  4. uk.co.mmscomputing.imageio.ppm:
    Classes that enable the java imageio system to read and write *.pbm (P4), *.pgm (P5), *.ppm (P6) files.
  5. uk.co.mmscomputing.imageio.sff:
    Classes that enable the java imageio system to read and write *.sff (structured fax format) files.
  6. uk.co.mmscomputing.imageio.tiff:
    Classes that enable the java imageio system to read and write *.tif (Tagged Image File Format ) files.
