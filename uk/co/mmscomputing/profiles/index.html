CAPI Profiles from various CAPI devices.

If you would like me to add your ISDN devices profile, just send me an email with the getProfile output.

  1. Eicon Diva Server Analog:
    Only analog modem card I know of.
  2. T-Eumex 820 LAN:
    PABX, Capi over LAN.

The answerphone application needs an adapter card that supports CAPI. All ISDN adapter cards for Windows and many (soon hopefully all) cards for Linux will support CAPI. The only analog or PSTN modem cards (I know of) that support CAPI are Eicon's Diva Server Analog cards.
Hence. You don't need an ISDN line anymore to use this software. In BT Land: In order to use Caller Line Identification you need to ask BT to enable the service: 'BT Privacy at Home'.

To download the whole uk.co.mmscomputing package including this website's documents please go to the download page.
