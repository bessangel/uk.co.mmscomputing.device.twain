AVM ISDN Monitor Applet
This applet works only with AVM ISDN Devices.
It is using a proprietary manufacturer request to poll the states of controller 1:
  1. Count of registered applications
  2. ISDN-Line Activity
  3. D-Channel Activity
  4. B-Channels Activity
uk.co.mmscomputing.application.avmisdnmonitor.jar

This is an applet that needs access to a native library (jcapi). The applet will only be able to access this library if you accept the security certificate it was signed with. But be aware: If you accept the certificate the applet has full access to your computer!
